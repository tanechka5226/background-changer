## Background Changer - VS Code Extension
Толстова Татьяна Владимровна группа М3114

Описание
- Плагин для Visual Studio Code, позволяющий применять пользовательские цветовые схемы для фона редактора кода. Позволяет легко переключаться между различными темами оформления.

Возможности
- Применение готовых цветовых тем для фона редактора

- Сброс цветов к стандартным настройкам VS Code

- Уведомления о результате применения темы
